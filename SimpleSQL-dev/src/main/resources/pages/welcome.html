Welcome to SimpleSQL!

Your lightweight database modeling tool.

About SimpleSQL

SimpleSQL is a lightweight SQL workbench, built in Java, that offers a streamlined interface for database management. With features like a customizable EER model editor, SQL query console, and project-based organization, simpleSQL makes it easy to design, manage, and query databases.

Use it to create, manage, and visualize database schemas without the complexity of enterprise-level tools.

Getting Started

Connect to your database, create a new project, and start building an EER model with SimpleSQL!

Compile your project and view the generated tables and sql code in our built-in editor.

To start a new project:

  • Navigate to "File" > "New" to create a project
  • Use the toolbar to add entities and relationships.
  • Compile your model to SQL with one click!

To open a project:

  • Navigate to "File" > "Open" to open a project
  • Use the toolbar to add entities and relationships.
  • Compile your model to SQL with one click!

Navigation Tips

SimpleSQL is designed for easy navigation. Here are some quick tips:

  • Use the toolbar for fast access to essential tools.
  • Right-click on the canvas to quickly add entities or relationships.
  • Use the scroll wheel to zoom in and out of your project view.
  • Pan the workarea by pressing the left mouse button and move around.
